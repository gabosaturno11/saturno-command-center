Interactive TOC Editor V2.0 STANDALONE

Voice-controlled Table of Contents editor with 20+ speech commands, 6 themes, gradient editor, and task management.

Stack: Pure vanilla JS, Web Speech API, CSS Variables, LocalStorage. Zero dependencies.

Key features: Voice commands ("expand all", "go to chapter 5", "dark theme"), debounced auto-save, keyboard shortcuts (Ctrl+S/E/R/M), priority task management.

Saturno Solar System VUE 3

Interactive visualization: 6 training planets (Foundation, Strength, Skills, Mobility, Power, Endurance), 385 exercises total, 10 research frameworks, 8 power skills with prerequisites.

Stack: Vue 3 + Vite. Build: npm run build → npx gh-pages -d dist

VB Command Center DOCS

Multi-Claude coordination hub. Contains COMMAND_CENTER.md (live sync doc), AOC_EXPANDED_TOC.md (7 Parts, 27 Chapters), and VB_SUBMISSION_PACKAGE/ (visuals + assets).

Purpose: Central reference for when multiple Claude instances work simultaneously on the book project.

SCC — State Schema

S = {
    tocs:               {},     // TOC versions (v1, v2) — v1 = 220 items canonical
    tocNotes:           {},     // Per-chapter notes from gradient panel
    research:           [],     // Research cards with A/B/C/D blocks
    decisions:          [],     // {id, date, title, context, status}
    bottlenecks:        [],     // {id, title, priority}
    mindmap:            {},     // {nodes:[], edges:[]} — canvas engine
    documents:          {},     // Per-chapter Quill content
    htmlFiles:          [],     // 26 entries: {name, path, tag, content, fileType}
    exercises:          [],     // Synced from matrixData
    matrixData:         {},     // {nodes:[], edges:[], enums:{}}
    stageNames:         {},     // Editable per-stage labels
    vault:              [],     // Knowledge Vault documents
    savedWorkouts:      [],     // Named saves: {name, items[], date}
    savedMindmaps:      [],     // Named mindmap saves
    currentWritingMode: '',     // Active voice mode
    _versions:          [],     // Last 10 state snapshots
    _tocCollapse:       {},     // TOC tree collapse state
    _matrixFilters:     {},     // Matrix checkbox persistence
    _wordTarget:        null,   // Writing word count target
    _compression:       50,     // Writing compression slider
    _technical:         50      // Writing technical depth slider
}
